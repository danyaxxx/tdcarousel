Options

items

Type: Number
Default: 4

The number of items you want to see on the screen.

nav

Type: Boolean
Default: false

Show next/prev buttons.

dots

Type: Boolean
Default: false

Show dots navigation.

smartSpeed

Type: Number
Default: 250

Speed Calculate.

responsive

Type: Object
Default: empty object

Object containing responsive options.

loop

Type: Boolean
Default: false

Go backwards when the boundary has reached.
